# ScannedIn
QR-based classroom attendance system.

A full-stack web application that allows professors to generate a unique QR code for each class session. Students scan the code, enter their details, and their attendance is automatically recorded. Professors can later view logs and export them.

## Features
- Generate a new QR code for each attendance session
- Students scan and seamlessly sign in from their device
- Automatically records name and student number
- Separate attendance log per session
- View past attendance records
- Export attendance list as PDF
- Simple and clean interface

## How It Works
1. Professor clicks **Log Attendance**
2. A unique QR code is generated
3. Students scan the QR code
4. Students enter their information
5. Attendance is saved to the session
6. Professor can view or export results
